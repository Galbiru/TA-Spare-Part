# OCR Sparepart Scanner

Project OCR berbasis Flask dan Tesseract OCR untuk mendeteksi sparepart seperti SSD, RAM, dan LCD menggunakan upload gambar maupun kamera realtime.

## Features
- OCR Image Scan
- OCR Camera Realtime
- Statistik Scan
- Export PDF & Excel
- Login Session
- MySQL Database

## Tech Stack
- Python Flask
- OpenCV
- Tesseract OCR
- Bootstrap 5
- MySQL
